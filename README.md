## ✨Project setup
### clone the project
```
git clone git@github.com:linong1526/vue-movies.git
```
### enter the project directory
```
### cd vue-movies
```
install dependency
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
